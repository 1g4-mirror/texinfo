19.6 Invoking pod2texi: Convert Pod to Texinfo

The pod2texi program translates Perl Pod documentation file(s) to Texinfo. There are two basic modes of operation: generating a standalone manual from each input Pod, or (if --base-level=1 or higher is given) generating Texinfo subfiles suitable for use with @include.

The pod2texi program may be useful outside of the rest of Texinfo, thus the invokation of pod2texi is documented using the man page format to follow the convention used in Perl standalone programs, with a version on the web http://www.gnu.org/software/texinfo/manual/pod2texi.html and a version included below. The version included in the manual is also an example of pod2texi use, as it is converted from Pod using pod2texi.
